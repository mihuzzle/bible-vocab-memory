# FI → EN Bible Vocabulary Match (5 Pairs)

Lightweight single-file matching game to practice translating Biblical vocabulary from Finnish to English.

## How it works
- Always **5 pairs** per round
- **Finnish on the left**, **English on the right**
- Tap one card on the left and one on the right to match
- Full text is always visible (no truncation)
- The board **scrolls**, so long phrases are readable on all screens

## Tech
- Single `index_simple.html`
- All data embedded (offline-ready)
- Designed for GitHub Pages and mobile browsers

## Data
Vocabulary source: *Uskonnollinen_sanasto__laaja.xls* (embedded at build time).

## Deploy
Upload `index_simple.html` to your GitHub Pages repo (rename to `index.html` if desired).
